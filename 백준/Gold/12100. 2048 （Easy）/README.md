##배운 점
파이썬에서도 C++의 포인터와 같은 개념이 들어있고,
배열의 사용에서 이에 대한 이해가 필요하다

처음에 복사 없이 brd로 변수를 통일했을 때는, 
move(n-1, movel(bd))
move(n-1, mover(bd))
연속으로 사용 시, movel이 bd를 반환만 해주는 게 아니라, bd자체의 값을 바꿔서, 아래의 bd에도 영향을 줬음,
얕은 복사와 깊은 복사에 대해 알게 됐고, 이를 사용해 위의 에러를 해결했다
